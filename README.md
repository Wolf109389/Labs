# Lab 2
## Рівень 3.
### Варіант 1 — Максимальна кількість хом’ячків

Зоомагазин продає хом’ячків, кожен з яких має:

- **H** — денну норму корму
- **G** — рівень жадібності (додатковий корм за кожного сусіда)

Якщо в одній клітці живе `k` хом’ячків, кожен з них з’їдає: H + G * (k - 1) пакетів їжі

Мета: знайти **максимальну кількість хом’ячків**, яку можна прогодувати при обмеженому запасі корму `S`.

Вхідні параметри:

- `S` — денний запас корму (0 ≤ S ≤ 10^9)
- `C` — кількість хом’ячків (1 ≤ C ≤ 10^5)
- `hamsters` — список `[H, G]` для кожного хом’ячка

---

## Алгоритм рішення

1. Використовується **бінарний пошук по відповіді**:
   - `left = 0`, `right = C`
   - Шукаємо найбільше `k`, яке можна прогодувати

2. Для кожного `k` обчислюємо необхідну кількість корму:
   - `food[i] = H[i] + G[i] * (k-1)`
3. Сортуємо масив `food` за допомогою **Merge Sort** (без готових сортерів) та беремо `k` найменших значень.

4. Перевіряємо, чи сума `k` найменших значень ≤ S:
   - Якщо так → збільшуємо `k` (`left = k + 1`)
   - Інакше → зменшуємо `k` (`right = k - 1`)

5. Результат — максимальне `k`.

---

## Приклади використання

```python
S = 7
C = 3
hamsters = [[1, 2], [2, 2], [3, 1]]
print(max_hamsters(S, C, hamsters))
# Результат: 2

S = 19
C = 4
hamsters = [[5, 0], [2, 2], [1, 4], [5, 1]]
print(max_hamsters(S, C, hamsters))
# Результат: 3

S = 2
C = 2
hamsters = [[1, 50000], [1, 60000]]
print(max_hamsters(S, C, hamsters))
# Результат: 1
```
